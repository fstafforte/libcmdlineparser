CommandLineParserThis class manages a process command line options.
It provides methods to add options, facilities to add
classic options such as 'version' or 'help' and to parse
a process command line, and get remaining arguments
that are not related to an option
More over provides methods to print out options usage
and the possibility to check if mandatory options
has been provided in the command line.
For each type of option a getter method is provided.
Sometimes user need to provide an option more then one time
in a process command line so there are getter methods that can be used
to get an array of options.
